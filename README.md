# hansung_minecraft_tail
마인크래프트의 컨텐츠 꼬리잡기를 플러그인 형식으로 제작해보았습니다
```mvn clean package```
를 터미널에 작성하시면
target폴더안에 tailtag-1.0.0.jar 파일이 생성될것입니다
이 파일을 마크 plugins 폴더에 넣으시면 적용 됩니다.
---
명령어 모음
---
/tail join 게임에 참가함
/tail start 게임 시작
/tail stop 게임 종료
/tail leave 게임 나가기
/tail help 명령어 모음
---
